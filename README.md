# ToyPromise

JavaScript Promise를 직접 구현해보는 학습용 프로젝트입니다.

## 설치

```bash
npm install
```

## 실행

```bash
# 테스트 실행
npm test

# TypeScript 빌드
npm run build
```

## 구현된 기능

- `then()` 메서드
- `catch()` 메서드
- `finally()` 메서드
- `ToyPromise.resolve()` 정적 메서드
- `ToyPromise.reject()` 정적 메서드
- 체이닝 지원
- thenable 평탄화
- 에러 전파
